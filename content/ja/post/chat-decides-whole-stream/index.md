「今日は何を配信しよう…」とタイトル画面で止まってしまうこと、ありますよね。
企画をゼロから考えるのは意外とエネルギーが要ります。
でも、その判断をチャットに委ねるだけで、準備の手間がぐっと減ります。

この記事で分かること：

- 丸投げ配信が視聴者との距離を縮める理由
- チャットが迷わない「3択プロンプト」の作り方
- 沈黙を防ぐ進行テンプレとフォローアップ術

## 企画を考えるのが面倒な日こそ「丸投げ」が効く

毎回凝った企画を用意する必要はありません。
「今日はチャットに決めてもらう」と宣言するだけで、配信に参加型のメリハリが生まれます。

視聴者側にも「自分の意見が配信に反映される」という没入感があります。
特に初見の人にとっては、「配信者と一緒に作る時間」という印象が残りやすくなります。

丸投げが効果を発揮しやすい場面を3つ挙げます。

1. **週末の夜など、雑談メインで見に来るリスナーが多い時間帯** — コメント数が多くなるぶん、選択肢への反応も集まりやすいです。
2. **大型企画の合間の「休息配信」として** — 前後に力を入れた配信があった週なら、あえてゆるく過ごすことでリズムが整います。
3. **体調やモチベが本調子ではない日** — 無理に構成を詰めず、チャットの流れに身を任せることで負担を減らせます。

## チャットに選んでもらう「3択プロンプト」の作り方

選択肢が多すぎるとチャットが分散して盛り上がりに欠けます。
基本は「3択」に絞るのがコツです。

プロンプトの作り方を3つのパターンで紹介します。

1. **話題セレクター型** — 「今日話すテーマ：A 昔の失敗談 / B 最近ハマったもの / C 視聴者への質問コーナー。チャットに A・B・C で教えて！」のように、どれを選んでも話せるネタを並べます。
2. **ゲームチョイス型** — 「手持ちのゲームから3本ピック → チャット投票 → 最多票をプレイ」という王道パターンです。候補を3本に絞っておけば、タイトル画面での迷いがなくなります。
3. **ミッション型** — 「配信中にやることを3つ提示 → チャットが順番を決める」という方式です。
「10分間早口クイズ → 苦手な歌を歌う → 真面目に感想を言う」など、軽めのミッションを並べると盛り上がります。

どのパターンでも、選択肢は配信開始の5〜10分前に決めておくのがおすすめです。
配信開始直後の「今から考えます」は視聴者を待たせてしまうため、事前に3つを書き出してから始めましょう。

## 進行テンプレ (開始5分 / 中盤40分 / 締め15分)

丸投げ配信でも、ある程度の時間配分を決めておくと安心です。
以下は1時間配信を想定したテンプレートです。

### 開始5分：導入と投票

「今日は〇〇をチャットに決めてもらいます！」と冒頭で宣言します。
続けて3つの選択肢を提示し、「A・B・C でチャットにお願いします」と呼びかけます。
2〜3分で結果を読み上げ、採用テーマに着手します。

### 中盤40分：採用テーマを深掘り

決まったテーマに沿って配信を進めます。
途中で「次の小ネタも追加で選んでもらう」と挟むと、中だるみを防げます。

中盤で使える小ネタの例を3つ挙げます。

1. **「5分ごとにチャットが質問を投げる」コーナー** — 視聴者からの質問に次々答える形式で、話題が尽きにくくなります。
2. **「BGMを3曲の中から選ぶ」** — これだけでチャットが活発になるため、沈黙対策としても有効です。
3. **「次にやることを再度3択で決める」** — 20分くらいで区切りをつけ、第2ラウンドの選択肢を出すと変化がつきます。

### 締め15分：振り返りと予告

「今日は〇〇を選んでくれてありがとうございました」と結果を振り返ります。
「次回は〇〇をやるかもしれません」など軽く予告を出すと、リピーターの滞在理由になります。

## 「丸投げ」が沈黙したときのフォローアップ

チャットが少ない時間帯や、初見が多い配信では、コメントが集まらないこともあります。
そういう時のために、3つの具体的な保険を用意しておきましょう。

1. **自分で1票を投じてスタートする** — 「自分はAに一票入れます。異論があればBかCで！」と最初に動くことで、視聴者がコメントしやすくなります。
2. **SNSで事前にアンケートを取っておく** — 配信前日に X で投票をしておきます。
「事前アンケートの結果はAが最多でした。配信内でもう一度確認します」と持ち込めば、ネタが枯れません。
3. **「コメントがなければ〇〇をやります」という罰ゲーム型** — 「5分以内に3コメントなければ酸っぱい飴を食べます」など、軽いペナルティを設定するとコメントを促せます。ハードルが高すぎない内容にするのがポイントです。

## まとめ

- 「今日はチャットに決めてもらう」と宣言するだけで、準備の手間が減り参加型の配信になります
- 3択プロンプトを事前に用意しておけば、開始直後からスムーズに投票を始められます
- コメントが少なくても自分で動いたり事前にアンケートを取ったりすれば、沈黙を回避できます

### 今日すぐ試せる1ステップ

次の配信で、開始前に3つの選択肢をメモに書き出してみてください。
「今日はチャットに決めてもらいます！ A・B・C で教えて」と冒頭で伝えるだけで、参加型配信がスタートします。
