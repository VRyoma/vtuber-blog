配信前に「あ、BGM 入れ忘れた」「OBS のシーン切り替え確認してない」と毎回違うものを忘れていませんか。

準備が安定しないと、配信開始までに焦ってしまい、本番中も「次は何を忘れるだろう」という不安が残ります。ミスを減らすには、自分の配信スタイルに合った**チェックリストを1枚持つ**のが一番の近道です。

この記事では、ChatGPT などの AI に配信フォーマットを伝えて、**自分専用のチェックリストを自動生成する方法**を解説します。プロンプト例をコピペするだけですぐに使えます。

**この記事で分かること:**
- 配信前の準備でよく起きる「抜け漏れ」の3パターン
- AI にチェックリストを出させるプロンプトの書き方
- 雑談・ゲーム・歌枠の配信タイプ別チェックリスト例

---

## 配信前の「抜け漏れ」が起きる 3 つのパターン

配信準備のミスには、ある程度パターンがあります。自分がどれに当てはまるかを知るだけで、対策が見えてきます。

### 1. やることは決まっているのに順序がバラバラ

OBS の起動、BGM の選曲、サムネの投稿、冒頭あいさつの確認——。やるべき項目は頭に入っていても、**毎回違う順番**で準備していませんか。

順序が決まっていないと、「あ、これまだだった」と途中で戻る手間が増えます。
5分で終わる準備が15分に伸びる原因の多くはここにあります。

### 2. 配信タイプごとに必要な作業が違う

雑談枠、ゲーム枠、歌枠で必要な準備は別物です。
ゲーム枠ならゲームの起動確認とキャプチャ設定、歌枠なら音源の並び順と音量バランスの確認が増えます。

どの枠でも同じチェックリストを使っていると、**不要な項目に紛れて大事な項目が埋もれて**しまいます。

### 3. 新しい企画を入れたときに限って基本を忘れる

コラボや新企画など「いつもと違うこと」に気を取られていると、マイクのミュート解除や配信キーの確認といった**基本作業を忘れがち**になります。

人は新しいことに集中すると、ルーティンを飛ばしやすくなります。だからこそ、基本を含めたチェックリストが役立ちます。

## AI にチェックリストを出させるプロンプトの型

ChatGPT などの AI に「配信準備のチェックリストを作って」と頼むだけでも、それなりのリストは出てきます。
しかし**自分の環境に合ったリスト**にするには、配信で使っているツールや配信フォーマットを具体的に伝えます。
たとえば「OBS Studio で YouTube 配信をしている」と一言添えるだけで、出力の精度が大きく変わります。

以下のプロンプトをコピーして、括弧の中身を自分用に書き換えて使ってください。

```
あなたは VTuber の配信アシスタントです。
以下の条件で、配信前の準備チェックリストを作成してください。

【配信環境】
- 配信ソフト: OBS Studio
- 配信プラットフォーム: YouTube
- 音声: VTube Studio + VoiceMeeter
- その他: Streamlabs（お知らせ通知）

【よくやる配信タイプ】
- 雑談枠（週3回程度）
- ゲーム実況枠（週1回）
- 歌枠（週1回）

【出力形式】
- 配信タイプごとに分ける
- 各項目にチェックボックス（[ ]）を付ける
- やるべき順に並べる
- 「配信開始30分前」「配信開始5分前」などタイミングで区切る
```

### プロンプトを自分用にカスタマイズするポイント

1. **配信環境** — 使っているツールを正確に書く。「OBS ではなく Streamlabs OBS」など微妙な違いも書くと、精度が上がります
2. **配信タイプ** — 自分が実際にやっている枠だけを書く。コラボ枠や企画枠があれば追加しましょう
3. **出力形式** — チェックボックス付きリストを指定すると、そのまま印刷やメモアプリに貼れます

AI が出力したリストは、最初は「たたき台」として使います。
実際に配信しながら「ここは不要だった」「この項目が足りない」を書き足していくと、徐々に自分専用のリストに育っていきます。

## 配信タイプ別のチェックリスト例 3 パターン

AI が出力するチェックリストのイメージを、配信タイプ別に示します。
以下は一例なので、自分の環境に合わせて書き換えて使ってください。

### 雑談枠

| タイミング | チェック項目 |
|---|---|
| 開始30分前 | YouTube Studio で配信を予約（タイトル・説明文・サムネ確認） |
| 開始30分前 | OBS 起動 → シーンが「待機」になっているか確認 |
| 開始30分前 | BGM の音量を再生して確認（ -25dB 前後が目安） |
| 開始15分前 | マイクのミュート解除 → 声が入るかテスト録画で確認 |
| 開始15分前 | 今日のトピックメモをデスクトップに置く |
| 開始5分前 | SNS に「まもなく配信開始」を投稿 |
| 開始5分前 | Streamlabs の通知表示が ON になっているか確認 |
| 開始直前 | 配信開始 → 10秒待ってから「待機」から「雑談」へ切替 |

### ゲーム実況枠

| タイミング | チェック項目 |
|---|---|
| 開始45分前 | ゲームを起動してアップデートがないか確認 |
| 開始30分前 | OBS のゲームキャプチャソースがゲーム画面を拾っているか確認 |
| 開始30分前 | ゲーム内の BGM とマイク音量のバランスをテスト録画で確認 |
| 開始15分前 | 配信フレームレートが安定しているか（OBS の統計で FPS 確認） |
| 開始15分前 | ゲームの UI 設定で配信に映したくない情報を非表示にする |
| 開始5分前 | 「まもなく開始」を SNS 投稿 |
| 開始直前 | 配信開始 → ゲーム画面が正しく映っているかコメント欄で確認 |

### 歌枠

| タイミング | チェック項目 |
|---|---|
| 開始45分前 | 歌う曲のリストを決めて並び順を確定 |
| 開始30分前 | 各曲の音源ファイルが正しく再生されるか確認 |
| 開始30分前 | OBS の音声ミキサーで「デスクトップ音声」と「マイク」の比率を調整 |
| 開始15分前 | 1曲目を流して音量バランスをテスト録画で確認 |
| 開始15分前 | 歌詞テロップまたは歌詞表示ツールの動作確認 |
| 開始5分前 | 喉のウォームアップ（発声練習・ストレッチ） |
| 開始直前 | 配信開始 → 最初の曲前のあいさつシーンに切替 |

## チェックリストを自分用に育てるコツ

AI が出したチェックリストは、そのまま使い続けるより**実際の配信経験を反映させて育てていく**と精度が上がります。

### 1. 配信後に「忘れたこと」をすぐメモする

配信が終わった直後に「今日忘れてたこと」を1行でもメモしておきましょう。
終了後のテンションが高いタイミングが、最も記憶が鮮明です。

メモした項目は、次回の配信前にチェックリストに追記します。
これを3〜4回繰り返すと、自分特有の抜け漏れがほぼ網羅されます。

### 2. 項目は「10〜15個」に絞る

チェックリストの項目が20個を超えると、確認自体が負担になります。
本当に重要な項目を**10〜15個**に絞り込みましょう。

絞り方の目安は「これを忘れると配信が成立しない」かどうかです。
「あると良い」レベルの項目は別リストに分けて、本番前は必須項目だけを回しましょう。

### 3. 定期的に AI に見直しを頼む

チェックリストを使い始めて1ヶ月ほど経ったら、もう一度 AI に投げてみましょう。
「現在のチェックリスト」と「最近よくやる配信」を伝えて、「不足している項目がないか」を聞くと、新しい視点が得られます。

配信環境が変わったとき（マイクを変えた、新しい配信プラットフォームを始めたなど）も、AI に環境を伝え直してリストを更新しましょう。

## まとめ

- 配信準備の抜け漏れは「順序のバラつき」「タイプ別の差異」「基本の忘れ」の3パターンに分類できる
- AI に配信環境とフォーマットを伝えるプロンプトを使えば、**自分専用のチェックリスト**を5分で生成できる
- リストは使いながら育てる。配信後のメモを反映させ、**10〜15項目に絞って**運用する

まずはこの記事のプロンプト例をコピーして、自分の配信環境に書き換えて AI に投げてみてください。
出てきたリストをそのまま次回の配信で使うだけで、準備の抜け漏れがぐっと減ります。

**今夜できるアクション:** プロンプトの「配信環境」を自分のツールに書き換えて AI に投げ、出てきたリストをメモアプリに貼る。配信前に横に置いてチェックを付けるだけで OK です。
