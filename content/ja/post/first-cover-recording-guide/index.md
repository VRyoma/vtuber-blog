「機材が揃ってから投稿しよう」と思い続けて、気づいたら 1 年経っていた——そういう話をよく聞きます。歌ってみたの最低限の機材はとてもシンプルで、今すぐ始められます。

この記事では、1 万円以内の機材構成と Audacity を使った録音手順を解説します。

**この記事で分かること**
- 初投稿に必要な最低限の機材と費用
- カラオケ音源の探し方と許諾確認
- DAW なしでできる録音〜書き出し手順

---

## 「機材が揃ってから」では永遠に始められない理由

完璧な機材を待ち続けると、環境が整っても「もっと良い機材があれば」というループに入ります。最初の投稿に必要なのは「聴こえる音」と「合っているピッチ」です。プロの音質は 2 本目・3 本目で追いかけて十分です。

初投稿の目的は「完成品を出す経験を積むこと」です。その経験なしに機材だけが増えても、投稿数は増えません。

---

## 1 万円以内の最低限構成

| 機材 | 具体例 | 費用目安 |
|---|---|---|
| コンデンサーマイク | FIFINE K669B など | 3,000〜5,000 円 |
| USB ケーブル（付属） | 多くのマイクに同梱 | 0 円 |
| ポップガード | クリップ式 | 500〜1,000 円 |
| 録音ソフト | Audacity（無料） | 0 円 |
| イヤホン（手持ちで可） | — | 0〜2,000 円 |

合計: 3,500〜8,000 円前後

USB マイクはドライバー不要で PC に直挿しできます。ファンタム電源も不要なので、オーディオインターフェースなしで録音を始められます。

### あると便利だが初回不要なもの

- オーディオインターフェース（XLR マイクを使うときに必要）
- 防音素材（クローゼットの中で録れば代用できる）
- DAW（GarageBand / Cubase など）

初回は Audacity だけで完結します。ミックスや音の調整を本格的にやりたくなったら、その時点で DAW を検討しましょう。

---

## カラオケ音源の探し方と許諾確認

歌ってみたには「カラオケ（オフボーカル）音源」が必要です。

### 音源の入手方法

| 方法 | 費用 | 注意点 |
|---|---|---|
| YouTube の公式カラオケ | 無料 | 動画として落とすのは規約違反 → 直接リンクのみ可 |
| ニコニコ動画のカラオケタグ | 無料 | 二次創作ガイドラインを確認する |
| JOYSOUND 公式カラオケ音源 | 一部有料 | 動画投稿ライセンス付き商品あり |
| 作曲者配布の音源 | 無料〜 | 個人制作曲は Booth / X で配布していることが多い |

**最もトラブルが少ないのは作曲者が直接配布している音源を使うことです。** 特に同人・インディーの楽曲はオフボーカル音源を公開している場合が多いです。

### 許諾確認の手順

1. 楽曲の公式サイト・作曲者の X を確認する
2. 「歌ってみた OK」「二次創作ガイドライン」を探す
3. 記載がない場合は 1 通連絡を送る

連絡なしに投稿した場合でも著作権の申請が来て収益が分配されるだけで済むケースはありますが、それが保証されているわけではありません。確認を省かないことを習慣にしましょう。

## Audacity を使った録音〜書き出し手順

### 事前準備

1. Audacity をインストールする（公式サイトから無料）
2. USB マイクを PC に接続する
3. Audacity を起動し「編集 → 設定 → オーディオデバイス」でマイクを選択する
4. カラオケ音源（MP3 または WAV）を手元に用意する

### 録音手順（5 ステップ）

```
1. カラオケ音源を Audacity にドラッグ&ドロップで読み込む
2. 再生しながらイヤホンで聴いて、音量バランスを確認する
3. 「R（録音）」ボタンを押して、マイクに歌いかける
4. 歌い終わったら「スペースキー」で停止する
5. 録音したトラックを確認する（波形が小さすぎる場合はマイク音量を上げる）
```

### 音量調整のポイント

| チェック項目 | 目安 |
|---|---|
| ボーカルの波形 | -6dB〜-3dB 程度（赤くならない範囲） |
| カラオケ音源の音量 | ボーカルより少し小さめ |
| 無音部分のノイズ | ほぼ平坦（波形がほとんど見えない）|

ノイズが気になる場合は「エフェクト → ノイズリダクション」で除去できます。無音部分を 1〜2 秒選択してからノイズプロファイルを取ると自動で処理されます。

### 書き出し手順

```
1. 「ファイル → 書き出し → MP3 として書き出す」を選ぶ
2. ビットレートは 320kbps を推奨
3. ファイル名を決めて保存
```

書き出したファイルをイヤホンで聴いて、カラオケとボーカルのバランスが自然かを確認します。

---

## 投稿前チェックリスト

```
□ 音量：ボーカルがカラオケに埋もれていない
□ ノイズ：無音部分にホワイトノイズが目立たない
□ ピッチ：音程が大きくズレているフレーズがない
□ 許諾：カラオケ音源の使用許諾を確認済み
□ サムネイル：タイトルと曲名が入っている
□ タイトル：「歌ってみた」「covered by 〇〇」の表記がある
```

全部チェックできたら投稿して OK です。完璧な仕上がりでなくても、投稿することが次の改善につながります。

---

## まとめ

- **最低限の機材は USB マイク 1 本と Audacity だけ**。1 万円以内で揃えられ、今日から録音を始められる
- **カラオケ音源は作曲者が配布している音源を使うのが最もシンプル**。許諾確認を習慣にすることがトラブル防止になる
- **録音〜書き出しは 5 ステップ**。まずチェックリストを通過したものを投稿することで、2 本目以降の改善点が具体的に見えてくる
