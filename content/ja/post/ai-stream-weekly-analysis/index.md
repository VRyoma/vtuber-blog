配信を続けているのに「何が機能していて、何が機能していないか」が分からないまま同じことを繰り返している——そういう状態になっていませんか。

YouTubeの分析ページを開いても数字が多すぎて、どこを見ればいいか分かりにくいです。AIに「この週のデータ」を渡すと、改善すべきポイントを絞り込んで教えてくれます。週15分の習慣にすると、1ヶ月で配信の質が目に見えて変わります。どんなデータを集めるか、プロンプトの使い方から解説します。

**この記事で分かること**
- 週次分析に必要なデータの集め方
- AIに渡すプロンプトの使い方
- 分析結果を次の配信に活かす方法

---

## 週次分析に必要なデータ

### YouTubeスタジオから取れる数字

週に1回、以下の数字をメモするかコピーします。YouTube スタジオの「アナリティクス」→「コンテンツ」で配信ごとの数字が確認できます。

**配信1本あたりで確認する項目：**
- 視聴回数（インプレッション数と区別する）
- 平均視聴時間（または平均視聴率）
- 高評価数
- コメント数
- チャット数（配信中のコメント数）

**チャンネル全体で確認する項目：**
- その週の総視聴時間
- 登録者の増減数
- インプレッションのクリック率

### 自分で記録する情報

数字だけでは分からない情報も記録しておきます。

- 配信したゲームまたはテーマ
- 配信時間帯と長さ
- 配信中に盛り上がった瞬間（印象に残ったコメントなど）
- 技術トラブルの有無

## AIで週次分析をするプロンプト

以下のプロンプトに今週のデータを貼り付けて使います。

```
私はVTuberで、自分の配信を週次で改善したいと思っています。
今週の配信データを分析して、改善点を教えてください。

【今週の配信データ】
配信1：[ゲーム名など] / 視聴回数:[N] / 平均視聴時間:[N分] / コメント数:[N] / 配信時間:[開始時刻] / 配信長さ:[N時間]
配信2：[ゲーム名など] / 視聴回数:[N] / 平均視聴時間:[N分] / コメント数:[N] / 配信時間:[開始時刻] / 配信長さ:[N時間]
（配信した本数分を並べる）

【チャンネル全体（今週）】
総視聴時間：[N時間]
登録者増減：[+N / -N]
インプレッションのクリック率：[N%]

【自分メモ】
今週よかった配信：[配信名]
今週の手応えがなかった配信：[配信名]
技術トラブル：[あり/なし。あれば内容]

以下の観点で分析してください：
1. 数字が良かった配信と悪かった配信の違い
2. 平均視聴時間が短い配信の原因として考えられること
3. 来週試すべき改善アクションを3つ（具体的に）
```

---

## 分析結果の見方と活かし方

### 「3つの改善アクション」をそのまま次週に試す

AIが提案する改善アクションは、具体的なものになります。例えば「配信開始の宣言を最初の30秒以内にする」「サムネのコピーにキャラクター名を入れる」などです。3つすべてを試すのではなく、1〜2つに絞って確実に実行します。

改善が多すぎると「何が効いたか分からない」になります。1つ試して結果を確認する方が、次週の分析に使えるデータが増えます。

### 「平均視聴時間」が最も重要な指標

視聴回数よりも平均視聴時間の方が配信の質を反映します。視聴回数が多くても平均視聴時間が1〜2分なら「タイトルはクリックされているが内容で離脱している」状態です。

平均視聴時間が伸びている配信に共通する要素をAIに聞くと、改善のヒントが出てきます。

```
プロンプト例：
「先週の分析で、[配信A]の平均視聴時間が[N分]で最も長かったです。
この配信に共通する要素から、視聴維持率を上げるために
私が毎回意識すべきことを3つ教えてください」
```

### 4週分溜まったら月次比較もできる

4週分のデータをAIに渡すと「先月と比べてどう変わったか」の月次比較もできます。

```
プロンプト例：
「4週間分のデータを渡します。月全体のトレンドとして
改善されている点・変わっていない点・悪化している点を
それぞれ1〜2個ずつ教えてください」
```

---

## 週次分析を習慣にするコツ

### 毎週同じ曜日・同じ時間にやる

「月曜の配信準備前に15分」のように固定します。気が向いたときにやる習慣は続きません。配信準備のルーティンに組み込むと、分析と準備がセットになります。

### スプレッドシートに週ごとのデータを貯める

Googleスプレッドシートに週次で数字を記入します。AIへのプロンプトに貼り付けるだけでなく、自分で数字の推移を視覚的に確認できます。4週分あれば月次レビューにも使えます。

### 「分析のための配信」はしない

改善を意識しすぎると「数字のために配信している」という感覚になりやすいです。分析は「楽しくやっている配信をもっと届けるための補助」です。数字が下がっても配信の内容自体が楽しければ続けることが最優先です。

---

## まとめ

- **YouTubeスタジオの数字をAIに渡すだけで改善点が絞れる**。視聴回数・平均視聴時間・コメント数と自分メモを週1回記録するだけでデータが揃う
- **分析の結果は「来週試す改善アクション3つ」に絞る**。多くを同時に試すと何が効いたか分からなくなる。1〜2つに絞って確認する
- **平均視聴時間を最優先の指標にする**。視聴回数よりも配信の質を反映しており、改善の方向性が見えやすい
