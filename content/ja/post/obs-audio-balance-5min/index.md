「ゲーム音が大きすぎて声が聞こえない」とコメントで言われたことはありませんか。

配信側ではバランスが合っていても、視聴者の環境では聞こえ方が変わることがあります。ゲーム音に声が負けていると、せっかくのトークが伝わらず、結果的に離脱につながります。

この記事では、OBS のオーディオミキサーを使って**音声ソースごとの目安 dB と優先度の付け方**を解説します。設定自体は5分程度で終わります。

**この記事で分かること:**
- マイク・ゲーム音・BGM それぞれの目安 dB
- OBS オーディオミキサーでの調整手順
- 配信後にアーカイブを1分聞いて確認する方法

---

## 音量バランスが崩れるとどうなるか — よくある3つの不満

配信の音量バランスは、配信者自身には気づきにくい問題のひとつです。ヘッドホンで確認していると「ちゃんと聞こえている」と思っても、視聴者側では別の聞こえ方をしていることがあります。

コメント欄や感想でよく寄せられる不満を3つ挙げます。

| 不満の内容 | 原因 | 影響 |
|---|---|---|
| 「ゲーム音が大きくて声が聞こえない」 | ゲーム音量がマイクより高い | トークが伝わらず、雑談メインの視聴者が離脱する |
| 「話しているときにBGMが邪魔」 | BGM の音量が高すぎる | 落ち着いた話の雰囲気が削がれる |
| 「声の大きさが途中で変わる」 | マイクとの距離が変化している | 聞き取りづらく、音割れの原因にもなる |

とくに「ゲーム音が大きすぎる」は、FPS やアクションゲームの配信で指摘されやすい傾向があります。銃声や爆発音が突発的に大きくなるタイトルでは、平均音量だけでは対応できないケースもあります。

## 音声ソースごとの目安 dB — マイク・ゲーム・BGM

音量バランスを整えるときは、**「何を一番聞かせたいか」を明確にする**ことが出発点です。VTuber 配信では、基本は「あなたの声」が一番聞こえる状態を目指します。

以下の表は、よく使われる3つの音声ソースの目安です。

| ソース | 目安 dB | 優先度 | 役割 |
|---|---|---|---|
| マイク（声） | -5 〜 0 dB | 最優先 | 配信のメインコンテンツ |
| ゲーム音 | -15 〜 -10 dB | 中 | 臨場感を支える伴奏 |
| BGM | -25 〜 -20 dB | 低 | 空白を埋める背景音 |

マイクが -5〜0 dB なのは、**声が配信の主役だから**です。ゲーム音は声の邪魔をしない範囲で、BGM は「話している間はほぼ意識されない」くらいの音量が目安です。

### OBS のボリュームスライダーと dB の関係

OBS のオーディオミキサーには 0〜100% のスライダーが表示されますが、内部的には dB 単位で管理されています。大まかな対応関係は以下の通りです。

| スライダー | おおよその dB |
|---|---|
| 100% | 0 dB（基準） |
| 70〜80% | -5〜-3 dB |
| 50% | -10 dB 前後 |
| 30% | -15〜-12 dB |
| 10〜20% | -25〜-18 dB |

あくまで目安なので、**ご自身の環境でテスト録画して耳で確認する**ことをおすすめめします。マイクの感度や部屋の反響具合で最適値は変わります。

## OBS オーディオミキサーの調整手順

目安 dB が分かったら、実際に OBS で調整します。以下の5ステップを順番に進めてください。

### ステップ1: オーディオミキサーを開く

OBS のメイン画面下部に「オーディオミキサー」というパネルがあります。マイク、デスクトップ音声（ゲーム音）、BGM などのソースが並んでいるはずです。ここが調整のメイン画面になります。

### ステップ2: マイクを一番大きくする

マイクのスライダーを 80〜100% に設定します。これが基準です。マイクの音量バーが緑〜黄色の範囲で安定して動くようにします。赤くなり続けている場合は音割れの危険があるので、少し下げてください。

### ステップ3: ゲーム音を下げて声が上回るようにする

ゲーム音のスライダーを 40〜50% 程度まで下げます。ここで実際にゲームを動かした状態で、マイクに向かって普通に話してみてください。**声がゲーム音よりはっきり聞こえる**位置まで下げたら OK です。

### ステップ4: BGM をほぼ意識されない音量にする

BGM のスライダーを 10〜20% 程度に設定します。話している間は BGM が「なんとなく流れている」くらい、雑談の間があいたときだけふわっと聞こえる程度が目安です。

### ステップ5: 30秒テスト録画して聞き返す

設定が終わったら、**30秒ほどテスト録画**します。ゲームを動かしながらマイクに話しかけ、その映像を再生して聞き返してください。この時、イヤホンで確認するとバランスの良し悪しが分かりやすくなります。

### 補足: 詳細設定プロパティを使うともっと細かく調整できる

OBS のオーディオミキサーの各ソースを右クリックすると「詳細設定プロパティ」が開けます。ここでは以下の設定が可能です。

- **音量（%）**: スライダーより細かい1%単位の調整
- **モニタリング**: 「モニターのみ」「モニターと出力」の切り替え
- **ダウンミックス**: ステレオ/モノラルの切り替え

ゲームの銃声だけ大きすぎる場合は、オーディオミキサーの「プロパティ」からゲーム音ソースに「リミッター」フィルターを追加すると、突発的な音の跳ね上がりを抑えられます。

## 配信後に確認する方法 — アーカイブを1分だけ聞く

設定が済んで本番の配信を終えたら、**アーカイブの真ん中あたりを1分だけ聞く**習慣をつけてみてください。これだけで音量バランスの問題に気づけることが多いです。

### 確認手順

1. アーカイブの中央付近（配信開始から30分経過したあたり）を開く
2. ゲームプレイ中のシーンを30秒聞く
3. 雑談シーンを30秒聞く
4. 声の聞こえ方と BGM のバランスをチェックする

### チェック項目

| チェック項目 | 判定基準 |
|---|---|
| 声がはっきり聞こえるか | ゲーム音やBGMに負けていないか |
| BGM が主張しすぎていないか | 話している間にBGMが気にならないか |
| 音割れしていないか | 大きな声を出したときに「バリバリ」というノイズがないか |
| 無音区間が不自然でないか | 完全に無音の時間が長く続いていないか |

イヤホンでの確認をおすすめします。スピーカーだと低音が響いてゲーム音が実際より小さく聞こえることがあり、バランスの判断が甘くなります。数百円のイヤホンでも、スピーカーより正確に音量バランスを把握できます。

問題を見つけたら、次回の配信前にオーディオミキサーで該当ソースのスライダーを調整してください。小さく変えすぎると今度は別の不満が出るので、**1回の調整は5%刻み**が目安です。

## まとめ

- **マイクを最優先**にし、ゲーム音・BGM は声の邪魔をしない音量に設定する
- OBS のオーディオミキサーで**マイク 80〜100% / ゲーム音 40〜50% / BGM 10〜20%** を出発点にする
- 配信後に**アーカイブを1分聞き返す**習慣で、継続的にバランスを改善する

次の配信の前に、オーディオミキサーのスライダーを見直して30秒テスト録画をしてみてください。マイクが一番大きいか、ゲーム音が声を上回っていないか、この2点だけでも確認する価値があります。
