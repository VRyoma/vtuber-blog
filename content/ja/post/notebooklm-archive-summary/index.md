3時間の配信アーカイブを見返して「面白かった箇所はどこか」を探すのは、もう一度3時間かけることになります。

Google の NotebookLM は、長文ドキュメントを読み込んで質問に答えてくれるAIツールです。配信の文字起こしをNotebookLMに入れると、様々な質問ができます。「今回の配信で盛り上がった場面はどこか」「切り抜き素材として使えそうな発言を3つ出して」などが代表例です。文字起こしの準備方法から、すぐ使えるプロンプト集まで解説します。

**この記事で分かること**
- NotebookLMの基本的な使い方
- 配信文字起こしを準備する手順
- 要約・ネタ抽出に使えるプロンプト集

---

## NotebookLMとは

Google が提供する無料のAIツールで、PDFやテキストなどのドキュメントを「ノートブック」に読み込ませて質問できます。2024年末に日本語対応が強化され、日本語の配信文字起こしにも使いやすくなっています。

一般的なチャットAIとの違いは、**読み込んだドキュメントの範囲内で答える**点です。「この配信でどんなことがあったか」という質問に対して、文字起こしの内容だけをもとに答えるため、AIが内容を捏造しにくい特徴があります。

---

## 配信の文字起こしを準備する

### YouTubeの自動字幕を使う

YouTubeにアップロードした配信アーカイブには、自動で字幕が生成されます。YouTubeスタジオから文字起こしを取得できます。

**手順：**
1. YouTubeスタジオ → 「動画」 → 対象の配信を選択
2. 「字幕」タブ → 「日本語（自動生成）」 → 「トランスクリプトをダウンロード」

ダウンロードしたファイルをメモ帳などで開くと、タイムスタンプ付きのテキストが取得できます。

### 文字起こしサービスを使う場合

YouTubeの自動字幕の精度が低い場合（早口・方言・ゲーム固有名詞が多い場合）は、文字起こしサービスを使います。「CLOVA Note」「Notta」などがVTuberの配信に比較的対応しています。無料プランで月数時間分は処理できます。

---

## NotebookLMへの読み込み方

1. [NotebookLM](https://notebooklm.google.com/) にGoogleアカウントでログイン
2. 「新しいノートブック」を作成
3. 「ソースを追加」→「テキストを貼り付け」または「ファイルをアップロード」
4. 文字起こしテキストを貼り付けるか、.txtファイルをアップロード

1つのノートブックに複数の配信文字起こしを入れることもできます。「先月の配信10本分」をまとめて入れると、横断的な質問が可能になります。「先月の配信全体を通じて視聴者の反応が良かった話題は何か」のような月次分析にも使えます。

## 要約・ネタ抽出に使えるプロンプト集

### 基本：配信全体の要約

```
この配信で起きた出来事を時系列で5点にまとめてください。
各項目にタイムスタンプがあれば一緒に示してください。
```

### 切り抜き・ショート動画の素材探し

```
この配信の中で、30秒〜2分のショート動画として切り抜けそうな場面を3つ選んでください。
理由と該当するタイムスタンプを教えてください。
```

### 盛り上がった場面の特定

```
配信中に視聴者の反応が特に良さそうな場面（笑い・驚き・感動など）を
3〜5個ピックアップしてください。
```

### 次の配信のネタ出し

```
この配信での視聴者のコメントや反応をもとに、
次回の配信で試すと面白そうなネタや企画を3つ提案してください。
```

### SNS投稿文の素材

```
この配信を見ていない人に「こんなことがあった配信でした」と伝える
X（Twitter）投稿文を140字以内で作ってください。
配信アーカイブへの興味を引く内容にしてください。
```

## 活用上の注意点

### 文字起こしの精度が結果に影響する

自動字幕の精度が低いと、NotebookLMの回答も不正確になります。固有名詞が多い配信（ゲームタイトル・スキル名など）は、文字起こし後にざっと見て明らかな誤字を直しておくと精度が上がります。

### 個人情報・センシティブな発言に注意する

NotebookLMはGoogleのサーバーに文字起こしをアップロードします。配信内で話した住所・電話番号などの個人情報が含まれていないか確認してから読み込みます。

### 無料プランのソース制限に注意する

2026年時点の無料プランは、ノートブック1つあたり最大50ソース・1ソースあたり最大50万トークンが上限です。長時間配信の場合は分割して読み込むか、特に重要な部分だけを抜粋します。

## まとめ

- **NotebookLMに文字起こしを読み込ませると、3時間のアーカイブを数分で要点整理できる**。切り抜き候補・SNS素材・次回ネタの抽出が質問1つでできる
- **文字起こしはYouTubeスタジオの字幕ダウンロード機能で取得できる**。精度が低い場合はCLOVA Noteなどのサービスをあわせて使う
- **プロンプトは「何に使うか」を明確にすると精度が上がる**。「切り抜き向け」「SNS投稿向け」など用途ごとに質問を変える
