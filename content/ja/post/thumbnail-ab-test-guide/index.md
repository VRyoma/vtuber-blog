「このサムネイルとこのサムネイル、どっちがいいと思う？」——視聴者に聞いても答えは出ません。サムネイルの良し悪しは「実際にクリックされた数」でしか分からないからです。

感覚でサムネイルを決め続けると、改善の根拠がなくなります。A/B テストを導入すると「どのパターンが数字として正しいか」が分かります。

**この記事で分かること**
- CTR（クリック率）が何%なら問題ないのか
- YouTube Studio を使った A/B テストの設定手順
- A/B テストができない場合の代替比較方法

---

## CTR の基準値と「改善が必要なライン」

YouTube Studio の「アナリティクス → リーチ」から、動画ごとの CTR（インプレッションのクリック率）を確認できます。

| CTR | 判断 |
|---|---|
| 10% 以上 | 優秀。サムネとタイトルが強い |
| 5〜10% | 平均的。改善余地がある |
| 2〜5% | 要改善。サムネかタイトルに問題がある可能性が高い |
| 2% 未満 | 優先的に見直す |

ただし CTR だけで判断しないことも重要です。CTR が高くても視聴維持率が低い（=内容と不一致）場合は、アルゴリズム評価が下がります。「クリックされて最後まで見てもらえる」サムネイルが理想です。

---

## YouTube の A/B テスト機能を使う手順

YouTube は一部アカウントに「サムネイルのテスト（A/B テスト）」機能を提供しています。

### 手順

```
1. YouTube Studio → 動画 → 対象の動画を開く
2. 「編集」→「サムネイル」→「テストを追加」
3. パターン A（現行）とパターン B（改善案）をアップロード
4. テスト期間（7〜14 日推奨）を設定して開始
5. 期間終了後に「クリック率の高いほうを選択」して確定
```

テスト期間中は YouTube が表示比率を自動で振り分けます。同じ視聴者層に届いた状態でどちらがクリックされたかを比較できます。

---

## A/B テスト機能が使えない場合の代替方法

### 方法 1：動画を同日に 2 本投稿して比較

同じ動画ではありませんが、似たコンテンツを 2 本異なるサムネイルで同じ時間帯に投稿し、1 週間後の CTR を比較します。完全な A/B テストではありませんが、傾向を掴む参考になります。

### 方法 2：同カテゴリーの過去動画と比較する

自分のチャンネルで似た動画を 5〜10 本抽出し、CTR が高い動画のサムネイルに共通点を探します。

```
確認ポイント：
- 顔の表情が見える/見えない
- テキストの文字数と配置
- 背景の明暗・コントラスト
- 数字が入っているかどうか
```

「自分のチャンネルでこの要素があると CTR が高い」というパターンが見えてきます。

### 方法 3：X でのインプレッション比較

X に 2 種類のサムネイル画像を「どちらが気になりますか？」の 1 択投票で投稿します。視聴者の直感的な反応をデータとして取得できます。厳密なテストではありませんが、0 から 24 時間で反応が得られます。

---

## サムネイル改善でまず試す 3 つの変更

何から変えればいいか迷った場合、次の 3 点から試します。

### 変更 1：顔の表情を大きく・鮮明にする

顔が小さすぎると表情が読めません。スマホのホーム画面サイズで見ても「どんな顔をしているか」が分かるサイズが目安です。感情が見える顔アップは CTR が上がりやすいです。

### 変更 2：テキストを「3〜5 文字」の核心だけにする

「今日のゲーム実況です！」のような説明テキストは不要です。「衝撃の結末」「1 時間詰んだ」のような「それだけで気になる 3〜5 文字」にします。

### 変更 3：背景のコントラストを上げる

暗いゲーム画面をそのまま使うとサムネが地味になります。明るい色の背景か、ゲーム画面に色補正をかけてコントラストを上げるだけで目立ちやすくなります。

---

## まとめ

- **CTR は 5%を下回ったら要改善**。YouTube Studio のリーチタブで確認して、2% 未満は最優先で見直す
- **A/B テスト機能が使えるなら 7〜14 日間テストして数字で決める**。感覚ではなくデータで選ぶことで改善が積み重なる
- **まず試す変更は「顔を大きく・テキストを 5 文字以内・コントラストを上げる」の 3 点**。1 つずつ変えてどれが効いたかを確認する
