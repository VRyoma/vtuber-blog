「配信のタイトルと説明文、毎回なんとなく書いている」という人は多いと思います。タイトルと説明文は、YouTubeの検索で発見されるかどうかを左右する重要な要素なのに、配信前後の忙しい時間に後回しにされがちです。

AIに任せる部分を決めておくと、1本あたり5分以内でSEOを意識したタイトルと説明文が作れます。

**この記事で分かること**
- タイトルと説明文がSEOに影響する理由
- AIに生成させるためのプロンプト
- 生成した文章をそのまま使える仕上げのコツ

---

## タイトルと説明文がSEOに影響する理由

### YouTube は検索エンジンとして使われている

YouTubeは動画プラットフォームであると同時に、月間数十億回の検索が行われる検索エンジンです。「モンハンワイルズ ボス攻略」「ホラーゲーム 実況」のように、ゲームタイトルや実況形式で検索されます。

タイトルと説明文にこれらのキーワードが含まれていると、検索結果に表示される可能性が高くなります。

### 説明文は2,000字まで書ける

YouTubeの動画説明文は最大5,000字書けますが、2,000字程度でも十分な効果があります。ゲームタイトル・実況者名・プレイ内容・関連キーワードを自然な文章で入れることで、検索でヒットする可能性が上がります。

---

## AIにタイトルを生成させるプロンプト

以下のプロンプトをそのまま使えます。`[ ]` 内を書き換えます。

```
以下の配信内容に対して、YouTube 検索で見つかりやすいタイトルを5パターン作ってください。

配信内容：[ゲームタイトルと実況した内容を1〜2文で]
実況スタイル：[初見プレイ / 攻略 / 縛りプレイ など]
ターゲット視聴者：[そのゲームを知っている人 / VTuberの実況が好きな人 など]

条件：
- 50字以内
- ゲームタイトルを含める
- 実況者名（[名前]）を含める
- 「初見」「攻略」などの検索されやすいワードを使う
```

---

## AIに説明文を生成させるプロンプト

```
以下の情報をもとに、YouTube動画の説明文を作ってください。

配信タイトル：[タイトル]
ゲームタイトル：[ゲーム名]
プレイ内容：[何をしたか、どこまで進んだかを2〜3文で]
チャンネル概要：[実況者名][チャンネルのジャンル・特徴]
SNSリンク：X→[URL]、Twitch→[URL]（あれば）

構成：
1. 動画の内容説明（3〜5文）
2. チャンネルの紹介（2〜3文）
3. SNS・チャンネル登録の案内
4. ハッシュタグ（#ゲームタイトル #VTuber #ゲーム実況）

自然な口語で書いてください。
```

---

## 生成した文章の仕上げ

### タイトルは1パターンだけ選んで微修正

AIが出した5パターンから最も内容に近いものを1つ選び、固有名詞や表現を自分の口調に合わせて1〜2語だけ直します。全部直す必要はありません。

### 説明文の冒頭2行を自分で書く

YouTubeの説明文は最初の2行が検索結果に表示されます。ここだけ自分で書き直すと、AIが生成した残りの文章を活かしながら自分らしさが出ます。

```
冒頭2行の例：
「モンハンワイルズのボス〇〇に挑戦しました！
初見でどこまで行けるか、見届けてください」
```

### ハッシュタグは末尾に3〜5個

説明文の末尾にゲームタイトル・実況スタイル・配信者名のハッシュタグを入れます。

```
#モンスターハンターワイルズ #ゲーム実況 #VTuber #初見プレイ #〇〇チャンネル
```

---

## まとめ

- **タイトルと説明文はYouTube検索の入口**。ゲームタイトルと実況スタイルのキーワードを含めると検索流入が生まれる
- **AIプロンプトに配信内容と条件を渡すと5分以内に原案が出る**。タイトルは5パターン生成して1つ選ぶ、説明文は冒頭2行だけ書き直す
- **説明文の冒頭2行が最重要**。検索結果のプレビューに表示される部分なので、ここだけは自分の言葉で書く
